IP Recon

Is dit IP-adres aan een VPN te koppelen?

Voer een IPv4- of IPv6-adres in. De check vraagt gelijktijdig twee onafhankelijke databases live op, omdat geen enkele bron volledig is.

Eigen API-keys gebruiken (aanbevolen)

Gratis account op proxycheck.io. Zet het domein waarop je deze pagina opent vrij onder de CORS-instelling in je dashboard (bare domein, zonder https:// of slash erachter).

Gratis account op vpnapi.io (1.000 checks/dag). Let op: vpnapi.io adviseert zelf om de key niet client-side te gebruiken — iedereen die de paginabron bekijkt kan 'm zien en jouw daglimiet verbruiken. Voor een privétool is dat meestal geen probleem.

Beide keys blijven alleen in dit browsertabblad en worden nergens opgeslagen.

proxycheck.iov2 API
vpnapi.iosecurity API

Geen enkele check is 100% sluitend — providers wisselen IP's en sommige VPN's gebruiken bewust residentiële adressen die niet als VPN herkenbaar zijn. Twee bronnen die elkaar tegenspreken is normaal; zie het als een indicatie, niet als bewijs.
